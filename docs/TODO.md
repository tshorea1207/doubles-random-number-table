# ダブルス乱数表アプリ 機能一覧（優先度順）

モバイル端末で使用するダブルス乱数表アプリに必要な機能を、優先度順に整理しました。

---

## 【必須機能】コア機能

| #   | 機能名                 | 説明                                                           | 実装状況           |
| --- | ---------------------- | -------------------------------------------------------------- | ------------------ |
| 1   | 参加者数の設定         | 4〜20人程度の範囲で柔軟に設定可能                              | ✅ (4〜16人)       |
| 2   | ラウンド数の設定       | 大会の規模・時間に応じて調整                                   | ✅ (1〜10ラウンド) |
| 3   | コート数の設定         | 同時進行できるコート数を指定                                   | ✅ (1〜4コート)    |
| 4   | 組み合わせ表の自動生成 | ペアリングと対戦相手の最適化                                   | ✅                 |
| 5   | ペア重複の最小化       | 同じ人と組む回数を均等に                                       | ✅                 |
| 6   | 対戦相手重複の最小化   | 同じ人と対戦する回数を均等に                                   | ✅                 |
| 7   | 生成結果の表示         | ラウンドごとの見やすい一覧表示                                 | ✅                 |
| 8   | 参加者名の入力・編集   | 番号だけでなく名前で管理                                       | ⬜                 |
| 9   | 固定ペアの指定         | 特定の2人を常にペアとして固定（夫婦、親子、初心者+経験者など） | ✅                 |
